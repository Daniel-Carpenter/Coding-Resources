# Generics

## What are Generics?

* Generic types are typically denoted as `T`, e.g. ArrayList<`T`>

## Types

|Type|Description|
|----|-----------|
|`K` |Key        |
|`N` |Number     |
|`T` |Type       |
|`V` |Value      |

## Wrapper Classes

* types (or `Classes`) that act like `primitive` data types, but are built in, such as `Integer` or `Character`, as apposed to `int` and `char, respectively.
